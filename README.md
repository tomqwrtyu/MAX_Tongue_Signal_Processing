# MAX-Tongue-repo
### Prerequisite
#### Python
This project is developed in Python 3.10.7.
<br>
<br>   `pip3 install -r requirement.txt` 
<br>
#### Socket IO
Make sure NodeJS(https://nodejs.org/en/) is downloaded and use below command. 
<br>
<br> `npm install express@4`
<br> `npm install socket.io`
<br>
